# 🎓 Student Performance Prediction using Machine Learning

## 📌 Project Overview

This project predicts students' overall academic performance using Machine Learning. It analyzes various academic factors such as study hours, attendance, and subject scores to estimate the overall score using a Linear Regression model.

The project demonstrates the complete Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), visualization, model training, evaluation, prediction, and model saving.

---

# 🎯 Project Objective

The objective of this project is to analyze student performance data and build a Machine Learning model that predicts students' overall scores based on important academic features.

---

# 📂 Dataset

Dataset Name:

**Student_Performance.csv**

Features used in this project:

- Study Hours
- Attendance Percentage
- Math Score
- Science Score
- English Score

Target Variable:

- Overall Score
